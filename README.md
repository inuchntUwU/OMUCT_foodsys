# OMUCT Food System (OMUCT_foodsys)

家庭のいまある食材の管理に使うことができるWebシステムです。

---

## 🌟 概要 (Overview)

`OMUCT_foodsys` は、フロントエンド技術（HTML/CSS/JavaScript）を用いて構築された食品管理・提供システムです。  
カメラなどによるUI提供し、リアルタイムな情報連携により食材ロス、家計負担の軽減を実現します。

---

## 🚀 主な機能 (Features)

* **🏠 ホーム画面 (`/home`)**
  * システムのポータルおよびメイン機能へのアクセス
* **📷 カメラ機能 (`/camera`)**
  * 画像認識やバーコード読み取り、食品の撮影・登録機能
* **🍳 食材表示画面 (`/kitchen`)**
  * 登録済みの食材をリスト形式で閲覧できます

---

### OMUCT Food System (OMUCT_foodsys)

家庭内の食材を一括管理し、使い切りをサポートすることで**食品ロス（フードロス）を削減するWebシステム**です。

---

## 🌟 システム概要 (Overview)

「せっかく買った食材を期限切れで捨ててしまった」「既にあるのに同じ食材を買ってしまった」といった家庭内の食品ロスを解消するためのアプリです。

カメラを使った手軽な食材の登録機能や、キッチンの状況に応じた在庫一覧・消費管理機能を提供し、日々の食材管理を無理なくスマートに行うことができます。

---

## 🚀 主な機能 (Features)

* **🏠 ホーム画面 (`/home`)**
  * システムのホーム
* **📷 カメラ登録機能 (`/camera`)**
  * 購入した食材や冷蔵庫内の撮影
* **🍳 キッチン・在庫管理画面 (`/kitchen`)**
  * 現在のストック食材一覧と賞味期限・消費期限の管理
